## How do I install these formulae?

`brew install tayheau/latexmk/<formula>`

Or `brew tap tayheau/latexmk` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "tayheau/latexmk"
brew "<formula>"
```
